# Prompt Templates

This directory contains example prompt templates used in the LangChain
examples. Prompt engineering often starts with carefully crafting
instructions and examples to guide the language model towards the
desired behaviour.

## Translation template

```
Translate the following phrase into French:

{phrase}
```

## Summarisation template

```
Summarise the following text in one sentence:

{text}
```

Feel free to add your own templates for classification, extraction and
other tasks.

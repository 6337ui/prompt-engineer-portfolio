# LangChain Examples

This folder demonstrates how to integrate LangChain into the broader
prompt engineer portfolio. The examples illustrate:

* Creating chains using prompt templates.
* Wrapping a custom HTTP-based LLM into a LangChain `LLM` class.
* Running asynchronous chains with user-provided text.

Refer to `chains/example_chain.py` for a simple summarisation chain and
`prompts/templates.md` for a few prompt templates that you can adapt
for your own use cases.
